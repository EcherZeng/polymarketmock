## 止损价格
假设 0.89 买入 UP 线：
有效入场价 = 0.89
全局止损价设为 0.69
价格跌到 0.69 就会自动卖出平仓

## 止损比例
假设 stop_loss_pct: 0.20（20% 相对止损）：

不管你买了多少份额（1份还是10000份），只要价格亏了20%，就立即卖出。

买 1000 元 → 价格跌 20% → 剩 800 元（亏 200 元）→ 自动卖
买 10000 元 → 价格跌 20% → 剩 8000 元（亏 2000 元）→ 自动卖

## 连续亏损降仓机制

假设：

stop_loss_pct: 0.20（20% 相对止损）
consecutive_loss_threshold: 3
loss_position_reduction_pct: 0.5

第1笔交易
买入 UP 0.89，预期卖点 0.712（亏20%）
实际在 0.71 卖出 → 亏损 ✗
计数器：1
第2笔交易
买入 DOWN 0.35，预期卖点 0.68（有效价 0.65，亏20%）
实际在 0.69 卖出 → 亏损 ✗
计数器：2
第3笔交易
买入 UP 0.75，预期卖点 0.60
实际在 0.59 卖出 → 亏损 ✗
计数器：3 触发降仓！
position_scale = 1.0 × (1 - 0.5) = 0.5
第4笔交易（降仓后）
原本想买 1000（所有可用余额） / 0.78 = 1282 份
降仓后只买 1282 × 0.5 = 641 份（买入份数减半）
